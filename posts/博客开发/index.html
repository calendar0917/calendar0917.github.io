准备工作

技术栈

后端

  • 核心框架：Spring Boot 2.x（简化配置，快速开发）
  • 持久层：MyBatis-Plus（在 MyBatis 基础上增强，提供 CRUD 简化）
  • 数据库：MySQL 8.0（存储博客核心数据）
  • 缓存：Redis（缓存热门文章、用户会话等）
  • 安全框架：Spring Security（处理认证授权）
  • API 文档：Swagger/OpenAPI（自动生成接口文档）
  • 部署：Docker（容器化部署，简化环境配置）

前端

  • 基础框架：Vue 3 + Vite（轻量高效的前端框架）
  • UI 组件库：Element Plus（与 Vue 3 配套的组件库）
  • 路由：Vue Router（前端路由管理）
  • 状态管理：Pinia（Vue 3 推荐的状态管理库）
  • HTTP 客户端：Axios（与后端 API 交互）
  • Markdown 编辑器：mavon-editor（博客编辑功能）

功能模块设计

核心功能

  1. 用户模块：注册、登录、个人信息管理
  2. 文章模块：发布、编辑、删除、查询、分类、标签
  3. 评论模块：文章评论、回复功能
  4. 分类与标签：文章归类和标签管理
  5. 首页展示：文章列表、热门推荐、最新发布

扩展功能（可选）

  1. 文章搜索（Elasticsearch）
  2. 阅读统计与排行榜
  3. 点赞与收藏功能
  4. 后台管理系统
  5. 图片上传（阿里云 OSS 等）

开发步骤

阶段 1：项目初始化与基础搭建

  1. 创建 Spring Boot 后端项目，配置 MySQL 和 Redis 连接
  2. 设计数据库表结构（用户表、文章表、分类表、标签表、评论表等）
  3. 创建 Vue 前端项目，配置基础路由和组件

阶段 2：核心功能开发

  1. 实现用户认证授权功能
  2. 开发文章 CRUD 接口及对应的前端页面
  3. 实现分类和标签管理功能
  4. 开发评论功能

阶段 3：优化与扩展

  1. 添加 Redis 缓存热门数据
  2. 实现分页和排序功能
  3. 开发后台管理界面
  4. 添加搜索功能

阶段 4：部署上线

  1. 项目打包与测试
  2. 配置 Docker 部署环境
  3. 上线到云服务器（如阿里云、腾讯云学生机）
